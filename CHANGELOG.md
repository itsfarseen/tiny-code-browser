# Changelog

## V1.1 - 2025-08-31

### Added
- Word wrapping toggle for the code viewer.
  - Implemented a button to switch between wrapped and unwrapped code.
  - State of word wrapping is persisted in local storage.

## V1.0 - 2025-08-16

### Added
- Initial release of the Tiny Code Browser.
- Basic file and directory browsing.
- Code content display.
- Theme toggle (light/dark mode).
- Font size adjustment for code viewer.
